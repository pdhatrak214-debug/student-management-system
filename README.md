# student-management-system
A Student Management System built with Python and Streamlit for managing students, teachers, grades, and academic records.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Register new students
- View student records
- Delete student records
- Email validation
- Unique roll number verification

### 👩‍🏫 Teacher Management
- Register teachers
- Manage staff records
- Delete teacher records
- Unique employee ID verification

### 📊 Grade Management
- Add grades for students
- Remove grades
- Calculate average marks
- Automatic grade classification (A, B, C, D, F)
- Performance overview dashboard

### 💾 Data Storage
- JSON-based data persistence
- Automatic saving and loading of records

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Object-Oriented Programming (OOP)
- JSON
- Abstract Base Classes (ABC)

## ⚡ Installation

### Clone the Repository

```bash
git clone https://github.com/pdhatrak214-debug/student-management-system.git
```

### Navigate to Project Folder

```bash
cd student-management-system
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
